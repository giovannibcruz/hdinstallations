Favicon Shape

How should the "HD" monogram be framed? Here's how each looks at actual favicon size (32px) and enlarged.

32px actual

A — Rounded Square

The modern standard. Matches how iOS, Android, and browsers render app icons. Comfortable, familiar, professional.

32px actual

B — Circle

Softer, more approachable. Works well in browser tabs but wastes corner space — letters need to be slightly smaller to fit.

32px actual

C — No Background

Just the letters, transparent background. Adapts to any browser theme (light/dark). Minimal and clean, but less distinctive in a crowded tab bar.
